Simulación de Distribuciones - TP2 (Mejorado)

Archivos:
- distributions.py: módulo con generadores de variables aleatorias.
- app_simulacion_tkinter.py: aplicación GUI en Tkinter.

Instrucciones:
1. Instala matplotlib y numpy si no las tienes:
   pip install matplotlib numpy

2. Ejecuta la aplicación:
   python app_simulacion_tkinter.py

3. Selecciona distribución, parámetros, tamaño de muestra e intervalos, luego 'Generar'.

4. Usa 'Limpiar' para reiniciar campos, gráfico y datos.
